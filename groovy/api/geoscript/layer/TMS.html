Package: geoscript.layer

[Groovy] Class TMS

        • geoscript.layer.TMS 

        • class TMS
extends ImageTileLayer

          A Tile Map Service (TMS) TileLayer. A TMS TileLayer can read or write Tiles if given a File directory but it is ready only if given a URL.

          Authors:
          Jared Erickson
