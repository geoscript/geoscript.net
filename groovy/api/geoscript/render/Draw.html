Package: geoscript.render

[Groovy] Class Draw

    • geoscript.render.Draw

    • Easily draw a geoscript.geom.Geometry, a geoscript.feature.Feature, and a geoscript.layer.Layer to an image or interactive App. 

       import static geoscript.render.Draw.*
 import geoscript.style.*
 import geoscript.geom.*
 draw(new Point(0,0).buffer(10),
    style: new Stroke("black",2) + new Fill("gray", 0.75),
    size: [250,250]
 )
