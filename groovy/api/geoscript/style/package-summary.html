Package geoscript.style

Package geoscript.style Description

GeoScript's Style API
  • Interface Summary 
    Interface Description
    Style The Style interface
  • Class Summary 
    Class Description
    ChannelSelection The ChannelSelection Symbolizer is used to style multi-spectral Rasters such as multi-band satellite imagery.
    ColorMap A ColorMap is a Symbolizer used to style Rasters by mapping pixel values to colors.
    Composite A Composite is a Symbolizer that contains one of more Symbolizers.
    ContrastEnhancement A ContrastEnhancement is a Symbolizer used to style Rasters by modifying the contrast.
    Fill A Symbolizer for area/polygonal geometries.
    Font A Symbolizer for a Label Font.
    Gradient The Gradient Composite Symbolizer creates gradients between a series of values and symbolizers or from values from Layer.
    Halo A Symbolizer for label background.
    Hatch A Symbolizer that repeats a pattern.
    Icon A Symbolizer for an external image or glyph.
    ImageOutline An ImageOutline Symbolizer applies a Stroke or Fill Symbolizer to a Raster.
    Label A Symbolizer for labeling a geometry.
    RasterSymbolizer A Raster Symbolizer is used to style Rasters.
    ShadedRelief Symbolize a Raster with a ShadedRelief or hill shading.
    Shape A Symbolizer for point geometries that consists of a color and size.
    Stroke A Symbolizer for linear geometries that consists of a color and a width.
    Symbolizer A Base class for all Symbolizers.
    Symbolizers The Symbolizers class is a collection of static methods that can be used to create Symbolizers.
    Transform A Symbolizer that wraps a geoscript.filter.Function generally used for transforming Geometry or String or Date formatting.
    UniqueValues The UniqueValues Composite creates a Symbolizer for each unique value from a Layer's Field.
