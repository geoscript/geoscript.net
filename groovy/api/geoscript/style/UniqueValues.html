[Groovy] Class UniqueValues

        • geoscript.style.UniqueValues

        • The UniqueValues Composite creates a Symbolizer for each unique value from a Layer's Field. 

           Layer layer = new Shapefile(file)
 UniqueValues sym = new UniqueValues(layer, "STATE_ABBR", "Greens")
