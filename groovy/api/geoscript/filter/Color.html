Package: geoscript.filter

[Groovy] Class Color

      • geoscript.filter.Color

      • A Color Expression and a set of Color Utilities used in the Style module. You can create a Color from an RGB String 

         new Color("0,255,0")

        or from a CSS color name: 

         new Color("silver")

        or from a hexadecimal: 

         new Color("#00ff00")

        or from a RGB List: 

         new Color([0,255,0])

        or from a RGB Map: 

         new Color([r: 0, g: 255, b: 0, a: 125])

        or from a HSL Map: 

         new Color([h: 0, s: 1.0, l: 0.5])

        Authors:
        Jared Erickson
