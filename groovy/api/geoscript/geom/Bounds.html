[Groovy] Class Bounds

    • geoscript.geom.Bounds

    • A Bounds is an Envelope with a geoscript.proj.Projection. 

       Bounds b = new Bounds(1,2,3,4, new geoscript.proj.Projection("EPSG:2927"))
