geom.Polygon()¶

class geom.Polygon(coords)¶
Arguments

  • coords (Array) – Coordinates array.

Create a new polygon.

Example Use¶

Sample code to new polygon:

>> var Polygon = require("geoscript/geom").Polygon;
>> var poly = Polygon([
..   [[-180, -90], [-180, 90], [180, 90], [180, -90], [-180, -90]],
..   [[-90, -45], [-90, 45], [90, 45], [90, -45], [-90, -45]]
.. ]);

Properties¶

In addition to the properties common to all geom.Geometry() subclasses, polygon geometries have the properties documented below.

Polygon.rectangle¶

Boolean This geometry is a rectangle.

Methods¶

Polygon geometries have the methods common to all geom.Geometry() subclasses.
